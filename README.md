# Desafios Python – Gabriel

Este repositório contém 3 programas em Python desenvolvidos para resolver os seguintes desafios:

---

#Desafio 1 — Cálculo de Comissões
Arquivo: `desafio1_comissoes.py`

- Lê a lista de vendas
- Aplica regras de comissão:
  - < 100 → 0%
  - < 500 → 1%
  - ≥ 500 → 5%
- Soma a comissão por vendedor
- Exibe o total de cada um

---

#✅ Desafio 2 — Movimentações de Estoque
Arquivo: `desafio2_estoque.py`

Permite registrar:
- Número identificador único
- Descrição da movimentação
- Entrada ou saída de estoque

E retorna o estoque final do produto.

---

#✅ Desafio 3 — Cálculo de Juros
Arquivo: `desafio3_juros.py`

Dado:
- valor
- data de vencimento

Calcula:
- juros de 2,5% ao dia de atraso
- total atualizado
